# Introduction 
Small lib to easily go from eg. 1w2d to either 216 (hours) or a datetime.timedelta and vice versa

# Getting Started
Requires no external modules
Build:
```
sudo python setup.py sdist bdist_wheel
twine upload dist/*
```
